# Earthquake-statistics-v1.0
지진 관련 정보 제공 웹 사이트 v1.0

## 주요 기능
* 세계 지진 발생 상황 실시간 제공
* 국내 지진 발생 기록 검색
* 국내 지진 추이를 규모별 및 지역별로 통계화·시각화
* 전국 지진 대피소 검색 
* Lists, Maps, and Statistics

## 개발 환경 · 사용 기술 · 언어
* Eclipse 
* Linux CentOS7
* Spring Framework 4.3.12
* Hadoop-2.9.1
* Java, JavaScript, Jquery, CSS, BootStrap
* Daum Maps API
* Google Chart API
